# 05-storage - 分布式存储

分布式存储系统与数据库技术。

## 子目录

- `dfs/` - 分布式文件系统（GFS, HDFS, Ceph）
- `graph/` - 图数据库
- `lakehouse/` - 数据湖架构与数据治理
- `newsql/` - NewSQL数据库（Spanner, TiDB, CockroachDB）
- `nosql/` - NoSQL数据库（Redis, MongoDB, Cassandra, HBase）
- `object-storage/` - 对象存储
- `replication/` - 数据复制策略与一致性哈希
- `search/` - 搜索引擎（Elasticsearch）
- `time-series/` - 时序数据库
- `wide-column/` - 宽列存储
- `governance/` - 数据治理
- `standards/` - 存储标准规范

## 核心内容

- 分布式存储选型
- 数据复制与一致性
- 存储性能优化
