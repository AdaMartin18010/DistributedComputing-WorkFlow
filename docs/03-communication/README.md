# 03-communication - 分布式通信

- `message-queue/` - 消息队列（Kafka, RabbitMQ, RocketMQ, Pulsar）
- `network/` - 分布式网络基础与时间同步

- `rpc/` - 远程过程调用
- `serialization/` - 序列化格式
