# 同态加密

## 概述

同态加密（Homomorphic Encryption, HE）是一种允许在密文上直接进行计算的加密技术。计算结果解密后与在明文上执行相同操作的结果一致。这一特性使得数据可以在加密状态下被处理，实现了"数据可用不可见"的理想隐私保护状态。

---

## 一、同态加密基础

### 1.1 同态性定义

```
同态加密的数学定义：

设加密函数为 E，解密函数为 D，明文空间为 M，密文空间为 C

加法同态：
    对于任意 m₁, m₂ ∈ M：
    D(E(m₁) ⊕ E(m₂)) = m₁ + m₂
    
    其中 ⊕ 是密文空间上的某种运算

乘法同态：
    对于任意 m₁, m₂ ∈ M：
    D(E(m₁) ⊗ E(m₂)) = m₁ × m₂
    
    其中 ⊗ 是密文空间上的某种运算

全同态（FHE）：
    同时支持任意次数的加法和乘法运算
    可以计算任意多项式函数

部分同态（PHE）：
    仅支持加法或乘法中的一种运算
    如：Paillier（加法）、RSA（乘法）

有些同态（SHE）：
    同时支持有限次数的加法和乘法
    如：BGV、BFV、CKKS方案
```

## 二、BFV方案详解

BFV（Brakerski-Fan-Vercauteren）是目前最流行的有些同态加密方案之一，支持整数运算。

### 2.1 BFV核心参数

```
BFV参数说明：

1. 多项式次数 n = 2^d
   • 通常是1024, 2048, 4096, 8192
   • 决定噪声容限和安全级别

2. 系数模数 q
   • 大整数，通常是多个小素数的乘积
   • 决定明文空间大小和乘法深度

3. 明文模数 t
   • 通常是2的幂或素数
   • 明文空间为 R_t = Z_t[x]/(x^n + 1)

4. 乘法深度 L
   • 可以执行的最大连续乘法次数
   • 由 q/t 的比例决定

参数选择示例（128位安全）：
┌──────────────┬──────────────┬──────────────┬──────────────┐
│   n (Poly)   │   q (Ciph)   │   t (Plain)  │   Depth      │
├──────────────┼──────────────┼──────────────┼──────────────┤
│    4096      │   109位      │    16位      │     2        │
│    8192      │   218位      │    16位      │     4        │
│   16384      │   438位      │    16位      │     8        │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

### 2.2 BFV基本操作

```python
# 注意：以下代码使用微软SEAL库的Python封装演示
# 实际安装：pip install seal-python 或 pip install pyseal

import numpy as np
from seal import (
    EncryptionParameters, scheme_type,
    SEALContext, KeyGenerator,
    Encryptor, Evaluator, Decryptor,
    BatchEncoder, CKKSEncoder,
    Plaintext, Ciphertext,
    GaloisKeys, RelinKeys
)

class BFVScheme:
    """
    BFV同态加密方案实现
    支持整数批量编码和计算
    """
    
    def __init__(self, poly_modulus_degree: int = 8192,
                 plain_modulus: int = 1032193):
        """
        初始化BFV方案
        
        Args:
            poly_modulus_degree: 多项式模次数（2的幂）
            plain_modulus: 明文模数
        """
        # 设置加密参数
        parms = EncryptionParameters(scheme_type.BFV)
        parms.set_poly_modulus_degree(poly_modulus_degree)
        
        # 设置系数模数（影响安全级别和乘法深度）
        # 使用推荐的系数模数
        parms.set_coeff_modulus(
            CoeffModulus.BFVDefault(poly_modulus_degree)
        )
        
        # 设置明文模数
        parms.set_plain_modulus(plain_modulus)
        
        # 创建上下文
        self.context = SEALContext(parms)
        
        # 密钥生成
        self.keygen = KeyGenerator(self.context)
        self.secret_key = self.keygen.secret_key()
        self.public_key = self.keygen.create_public_key()
        
        # 创建运算对象
        self.encryptor = Encryptor(self.context, self.public_key)
        self.evaluator = Evaluator(self.context)
        self.decryptor = Decryptor(self.context, self.secret_key)
        
        # 批量编码器（支持SIMD操作）
        self.batch_encoder = BatchEncoder(self.context)
        self.slot_count = self.batch_encoder.slot_count()
        
        print(f"BFV Scheme initialized")
        print(f"  Polynomial modulus degree: {poly_modulus_degree}")
        print(f"  Slot count: {self.slot_count}")
    
    def encode(self, data: list) -> Plaintext:
        """
        编码明文数据
        
        Args:
            data: 整数列表，长度不超过slot_count
        """
        # 填充到slot_count
        padded = data + [0] * (self.slot_count - len(data))
        
        plain = Plaintext()
        self.batch_encoder.encode(padded, plain)
        return plain
    
    def decode(self, plain: Plaintext) -> list:
        """解码明文"""
        result = []
        self.batch_encoder.decode(plain, result)
        return result
    
    def encrypt(self, plain: Plaintext) -> Ciphertext:
        """加密明文"""
        encrypted = Ciphertext()
        self.encryptor.encrypt(plain, encrypted)
        return encrypted
    
    def decrypt(self, encrypted: Ciphertext) -> Plaintext:
        """解密密文"""
        plain = Plaintext()
        self.decryptor.decrypt(encrypted, plain)
        return plain
    
    def add(self, ct1: Ciphertext, ct2: Ciphertext) -> Ciphertext:
        """
        密文加法
        result = ct1 + ct2
        """
        result = Ciphertext()
        self.evaluator.add(ct1, ct2, result)
        return result
    
    def add_plain(self, ct: Ciphertext, pt: Plaintext) -> Ciphertext:
        """
        密文与明文相加
        result = ct + pt
        """
        result = Ciphertext()
        self.evaluator.add_plain(ct, pt, result)
        return result
    
    def multiply(self, ct1: Ciphertext, ct2: Ciphertext) -> Ciphertext:
        """
        密文乘法
        result = ct1 * ct2
        
        注意：乘法后密文大小增加，需要重线性化
        """
        result = Ciphertext()
        self.evaluator.multiply(ct1, ct2, result)
        return result
    
    def multiply_plain(self, ct: Ciphertext, pt: Plaintext) -> Ciphertext:
        """
        密文与明文相乘
        result = ct * pt
        """
        result = Ciphertext()
        self.evaluator.multiply_plain(ct, pt, result)
        return result
    
    def relinearize(self, ct: Ciphertext, relin_keys: RelinKeys) -> Ciphertext:
        """
        重线性化
        将密文大小减小到最小，减少后续计算开销
        """
        result = Ciphertext()
        self.evaluator.relinearize(ct, relin_keys, result)
        return result
    
    def rotate_rows(self, ct: Ciphertext, shift: int, 
                    gal_keys: GaloisKeys) -> Ciphertext:
        """
        行旋转（SIMD操作）
        将密文中的槽位循环左移shift个位置
        """
        result = Ciphertext()
        self.evaluator.rotate_rows(ct, shift, gal_keys, result)
        return result


# 使用示例
if __name__ == "__main__":
    # 初始化BFV方案
    bfv = BFVScheme(poly_modulus_degree=8192)
    
    # 生成重线性化密钥和Galois密钥
    relin_keys = bfv.keygen.create_relin_keys()
    gal_keys = bfv.keygen.create_galois_keys()
    
    # 示例1：批量加密和加法
    print("\n=== Example 1: Batch Encryption and Addition ===")
    
    data1 = [1, 2, 3, 4, 5]
    data2 = [10, 20, 30, 40, 50]
    
    # 编码和加密
    plain1 = bfv.encode(data1)
    plain2 = bfv.encode(data2)
    ct1 = bfv.encrypt(plain1)
    ct2 = bfv.encrypt(plain2)
    
    # 同态加法
    ct_sum = bfv.add(ct1, ct2)
    
    # 解密和解码
    plain_result = bfv.decrypt(ct_sum)
    result = bfv.decode(plain_result)
    
    print(f"Data 1: {data1}")
    print(f"Data 2: {data2}")
    print(f"Encrypted Sum (first 5): {result[:5]}")
    
    # 示例2：密文与明文相乘
    print("\n=== Example 2: Ciphertext-Plaintext Multiplication ===")
    
    scalar = bfv.encode([2] * bfv.slot_count)
    ct_scaled = bfv.multiply_plain(ct1, scalar)
    
    plain_result = bfv.decrypt(ct_scaled)
    result = bfv.decode(plain_result)
    print(f"Data 1 * 2 (first 5): {result[:5]}")
    
    # 示例3：密文乘法
    print("\n=== Example 3: Ciphertext Multiplication ===")
    
    ct_product = bfv.multiply(ct1, ct2)
    ct_product = bfv.relinearize(ct_product, relin_keys)
    
    plain_result = bfv.decrypt(ct_product)
    result = bfv.decode(plain_result)
    print(f"Data 1 * Data 2 (first 5): {result[:5]}")
```

## 三、CKKS方案（浮点数同态加密）

CKKS方案支持浮点数的近似计算，特别适合机器学习应用。

```python
class CKKSScheme:
    """
    CKKS同态加密方案
    支持浮点数向量的近似计算
    """
    
    def __init__(self, poly_modulus_degree: int = 8192,
                 scale: float = 2**40):
        """
        Args:
            poly_modulus_degree: 多项式次数
            scale: 编码缩放因子，影响精度
        """
        parms = EncryptionParameters(scheme_type.CKKS)
        parms.set_poly_modulus_degree(poly_modulus_degree)
        
        # CKKS使用特定链式系数模数
        parms.set_coeff_modulus(CoeffModulus.Create(
            poly_modulus_degree,
            [60, 40, 40, 60]  # 素数位数链
        ))
        
        self.context = SEALContext(parms)
        self.scale = scale
        
        # 密钥生成
        self.keygen = KeyGenerator(self.context)
        self.secret_key = self.keygen.secret_key()
        self.public_key = self.keygen.create_public_key()
        
        # 运算对象
        self.encryptor = Encryptor(self.context, self.public_key)
        self.evaluator = Evaluator(self.context)
        self.decryptor = Decryptor(self.context, self.secret_key)
        
        # CKKS编码器
        self.encoder = CKKSEncoder(self.context)
        self.slot_count = self.encoder.slot_count()
    
    def encode(self, data: list) -> Plaintext:
        """编码浮点向量"""
        plain = Plaintext()
        self.encoder.encode(data, self.scale, plain)
        return plain
    
    def decode(self, plain: Plaintext) -> list:
        """解码"""
        result = []
        self.encoder.decode(plain, result)
        return result
    
    def encrypt(self, plain: Plaintext) -> Ciphertext:
        """加密"""
        encrypted = Ciphertext()
        self.encryptor.encrypt(plain, encrypted)
        return encrypted
    
    def decrypt(self, encrypted: Ciphertext) -> Plaintext:
        """解密"""
        plain = Plaintext()
        self.decryptor.decrypt(encrypted, plain)
        return plain
    
    def add(self, ct1: Ciphertext, ct2: Ciphertext) -> Ciphertext:
        """密文加法"""
        result = Ciphertext()
        self.evaluator.add(ct1, ct2, result)
        return result
    
    def multiply(self, ct1: Ciphertext, ct2: Ciphertext) -> Ciphertext:
        """
        密文乘法
        CKKS乘法后需要rescale
        """
        result = Ciphertext()
        self.evaluator.multiply(ct1, ct2, result)
        self.evaluator.rescale_to_next_inplace(result)
        return result
    
    def rescale(self, ct: Ciphertext) -> Ciphertext:
        """
        Rescale操作
        减少scale并模切换
        """
        result = Ciphertext()
        self.evaluator.rescale_to_next(ct, result)
        return result


# CKKS应用：加密向量点积
def encrypted_dot_product(ckks: CKKSScheme, 
                          vec1: Ciphertext, 
                          vec2: Ciphertext,
                          gal_keys: GaloisKeys) -> Ciphertext:
    """
    计算加密向量的点积
    
    算法：
    1. 逐元素相乘
    2. 旋转并累加所有槽位
    """
    # 逐元素相乘
    product = ckks.multiply(vec1, vec2)
    
    # 累加所有槽位（类似于reduce sum）
    result = product
    for i in range(int(np.log2(ckks.slot_count))):
        rotated = Ciphertext()
        ckks.evaluator.rotate_vector(result, 2**i, gal_keys, rotated)
        ckks.evaluator.add_inplace(result, rotated)
    
    return result
```

```python
class PrivateLogisticRegression:
    """
    隐私保护逻辑回归（简化示意）
    使用同态加密保护模型训练和推理
    """
    
    def __init__(self, ckks: CKKSScheme):
        self.ckks = ckks
        self.weights = None
    
    def encrypt_model(self, weights: np.ndarray) -> Ciphertext:
        """加密模型权重"""
        plain = self.ckks.encode(weights.tolist())
        return self.ckks.encrypt(plain)
    
    def encrypted_predict(self, 
                         encrypted_weights: Ciphertext,
                         encrypted_input: Ciphertext,
                         gal_keys: GaloisKeys) -> Ciphertext:
        """
        加密预测
        
        计算：sigmoid(w · x) ≈ 0.5 + 0.197 * (w · x) （低阶近似）
        """
        # 计算点积
        dot = encrypted_dot_product(
            self.ckks, encrypted_weights, encrypted_input, gal_keys
        )
        
        # 应用sigmoid近似（多项式近似）
        # sigmoid(x) ≈ 0.5 + 0.197x - 0.004x^3
        
        # 缩放因子
        scale_factor = self.ckks.encode([0.197] * self.ckks.slot_count)
        
        result = self.ckks.multiply_plain(dot, scale_factor)
        
        # 加上0.5偏移
        offset = self.ckks.encode([0.5] * self.ckks.slot_count)
        result = self.ckks.ckks.add_plain(result, offset)
        
        return result


# 安全多方计算中的同态加密
class SecureAggregation:
    """
    安全聚合：多个参与方在加密状态下贡献数据
    """
    
    def __init__(self, bfv: BFVScheme):
        self.bfv = bfv
        # 所有参与方使用相同的公钥
        self.public_key = bfv.public_key
    
    def contribute(self, data: list) -> Ciphertext:
        """
        参与方贡献加密数据
        """
        plain = self.bfv.encode(data)
        return self.bfv.encrypt(plain)
    
    def aggregate(self, contributions: list) -> Ciphertext:
        """
        聚合所有贡献（同态加法）
        
        result = contrib1 + contrib2 + ... + contribN
        """
        if not contributions:
            raise ValueError("No contributions")
        
        result = contributions[0]
        for contrib in contributions[1:]:
            result = self.bfv.add(result, contrib)
        
        return result
    
    def get_result(self, encrypted_result: Ciphertext) -> list:
        """只有私钥持有者可以解密"""
        plain = self.bfv.decrypt(encrypted_result)
        return self.bfv.decode(plain)
```

## 五、性能优化与最佳实践

### 5.1 性能考虑

```
同态加密性能特点：

操作延迟（粗略估计，取决于硬件和参数）：
┌─────────────────────────────────────────────────────────────┐
│  操作              │  延迟（BFV/CKKS，现代CPU）             │
├─────────────────────────────────────────────────────────────┤
│  密钥生成          │  10-100 ms                             │
│  加密              │  1-10 ms                               │
│  解密              │  0.5-5 ms                              │
│  密文-密文加法      │  < 1 ms                                │
│  密文-密文乘法      │  5-50 ms                               │
│  重线性化          │  5-30 ms                               │
│  旋转              │  5-30 ms                               │
│  自举（FHE）       │  100ms - 1s（主要开销）                 │
└─────────────────────────────────────────────────────────────┘

密文大小（以BFV为例）：
• 8192次多项式，218位系数模数
• 单个密文约 256 KB
• 对比：原始明文约 16 KB（8192个16位整数）

优化策略：
1. 使用批量编码（SIMD）同时处理多个值
2. 选择合适的参数（在安全性和性能间权衡）
3. 预计算和缓存密钥
4. 使用GPU加速（如cuFHE）
5. 减少乘法深度，使用加法替代
```

## 六、安全考虑

1. **参数选择**：使用标准参数（如SEAL默认），避免自行设计
