## 📋 执行摘要

联邦学习（Federated Learning, FL）是一种分布式机器学习范式，允许多个参与方在不共享原始数据的前提下协作训练共享模型，通过本地计算和参数聚合实现隐私保护和数据合规。

---

## 一、核心概念

### 1.1 定义与原理

**联邦学习**由Google于2016年提出，核心思想是：
> "将代码带给数据，而不是将数据带给代码"

**核心原理**：

1. **本地训练**：各参与方在本地私有数据上训练模型
2. **参数上传**：仅上传模型参数（或梯度），不上传原始数据
3. **安全聚合**：中央服务器聚合各方参数更新全局模型
4. **模型下发**：将更新后的全局模型分发回各参与方

### 1.2 关键特性

- **数据隐私保护**：原始数据不出本地
- **数据异构性**：各参与方数据分布不一致（Non-IID）
- **通信效率**：只传输模型参数，大幅减少通信量
- **参与方异构性**：设备能力、网络条件差异大
- **去中心化可选**：支持有中心或无中心架构

### 2.2 算法原理

#### 2.2.1 FedAvg算法

**Federated Averaging**是最经典的联邦学习聚合算法。

**算法描述**：

**输入**：

- K个客户端，每轮选择比例C
- 本地训练轮数E，本地batch大小B
- 学习率η，全局模型w_0

**输出**：优化后的全局模型w_T

**步骤**：

```
对于每轮通信 t = 1, 2, ..., T:
    1. 服务器随机选择 m = max(C·K, 1) 个客户端集合S_t

    2. 对于每个客户端k ∈ S_t并行执行:
       a. 从服务器接收全局模型 w_t
       b. 本地训练: w_k = LocalTrain(k, w_t)
          对于每个本地epoch i = 1..E:
              对于batch b ∈ 本地数据划分:
                  w_k ← w_k - η∇L(w_k; b)
       c. 计算模型更新: Δw_k = w_k - w_t
       d. 上传Δw_k到服务器

    3. 服务器聚合:
       w_{t+1} = w_t + Σ_{k∈S_t} (n_k/n) · Δw_k
       其中 n_k 是客户端k的样本数, n = Σ n_k
```

**Python实现**：

```python
import torch
import torch.nn as nn
from typing import List, Dict, Tuple
import copy

class FederatedServer:
    """联邦学习服务器"""

    def __init__(self, model: nn.Module, fraction_clients: float = 0.1):
        self.global_model = model
        self.fraction_clients = fraction_clients
        self.client_weights = {}

    def select_clients(self, num_clients: int) -> List[int]:
        """随机选择参与本轮训练的客户端"""
        m = max(int(self.fraction_clients * num_clients), 1)
        selected = torch.randperm(num_clients)[:m].tolist()
        return selected

    def aggregate_fedavg(self,
                         client_updates: List[Tuple[int, Dict]]) -> Dict:
        """
        FedAvg聚合算法

        Args:
            client_updates: [(client_id, state_dict), ...]

        Returns:
            聚合后的全局模型状态
        """
        global_dict = self.global_model.state_dict()
        aggregated = copy.deepcopy(global_dict)

        # 计算总样本数
        total_samples = sum(self.client_weights.get(cid, 1)
                           for cid, _ in client_updates)

        # 初始化聚合结果
        for key in aggregated.keys():
            aggregated[key] = torch.zeros_like(aggregated[key])

        # 加权聚合
        for client_id, client_state in client_updates:
            weight = self.client_weights.get(client_id, 1) / total_samples
            for key in aggregated.keys():
                aggregated[key] += weight * client_state[key]

        return aggregated

    def train_round(self, clients: List['FederatedClient']) -> float:
        """执行一轮联邦学习"""
        # 1. 选择客户端
        selected_ids = self.select_clients(len(clients))
        selected_clients = [clients[i] for i in selected_ids]

        # 2. 下发全局模型
        global_state = copy.deepcopy(self.global_model.state_dict())

        # 3. 并行本地训练
        client_updates = []
        total_loss = 0.0

        for client in selected_clients:
            client_state, loss = client.local_train(global_state)
            client_updates.append((client.client_id, client_state))
            total_loss += loss
            # 记录客户端样本权重
            self.client_weights[client.client_id] = len(client.dataset)

        # 4. 聚合更新
        aggregated_state = self.aggregate_fedavg(client_updates)
        self.global_model.load_state_dict(aggregated_state)

        return total_loss / len(selected_clients)


class FederatedClient:
    """联邦学习客户端"""

    def __init__(self,
                 client_id: int,
                 model: nn.Module,
                 dataset: torch.utils.data.Dataset,
                 local_epochs: int = 5,
                 batch_size: int = 32,
                 lr: float = 0.01):
        self.client_id = client_id
        self.model = model
        self.dataset = dataset
        self.local_epochs = local_epochs
        self.batch_size = batch_size
        self.lr = lr
        self.dataloader = torch.utils.data.DataLoader(
            dataset, batch_size=batch_size, shuffle=True
        )

    def local_train(self, global_state: Dict) -> Tuple[Dict, float]:
        """
        本地训练

        Returns:
            (本地模型状态, 平均损失)
        """
        # 加载全局模型
        self.model.load_state_dict(global_state)
        self.model.train()

        optimizer = torch.optim.SGD(self.model.parameters(), lr=self.lr)
        criterion = nn.CrossEntropyLoss()

        total_loss = 0.0
        num_batches = 0

        # 本地多轮训练
        for epoch in range(self.local_epochs):
            for batch_idx, (data, target) in enumerate(self.dataloader):
                optimizer.zero_grad()
                output = self.model(data)
                loss = criterion(output, target)
                loss.backward()
                optimizer.step()

                total_loss += loss.item()
                num_batches += 1

        avg_loss = total_loss / num_batches
        return copy.deepcopy(self.model.state_dict()), avg_loss
```

#### 2.2.2 改进聚合算法

**FedProx**（处理Non-IID数据）：

```python
class FedProxClient(FederatedClient):
    """FedProx客户端 - 添加近端正则项"""

    def __init__(self, *args, mu: float = 0.01, **kwargs):
        super().__init__(*args, **kwargs)
        self.mu = mu  # 近端项系数

    def local_train(self, global_state: Dict) -> Tuple[Dict, float]:
        self.model.load_state_dict(global_state)
        self.model.train()

        # 保存全局模型用于近端项
        global_params = [p.clone().detach() for p in self.model.parameters()]

        optimizer = torch.optim.SGD(self.model.parameters(), lr=self.lr)
        criterion = nn.CrossEntropyLoss()

        for epoch in range(self.local_epochs):
            for data, target in self.dataloader:
                optimizer.zero_grad()
                output = self.model(data)
                loss = criterion(output, target)

                # 添加近端正则项: μ/2 * ||w - w_global||^2
                proximal_term = 0.0
                for param, global_param in zip(self.model.parameters(), global_params):
                    proximal_term += torch.norm(param - global_param) ** 2

                loss += (self.mu / 2) * proximal_term
                loss.backward()
                optimizer.step()

        return copy.deepcopy(self.model.state_dict()), loss.item()
```

**FedNova**（处理异构本地步数）：

```python
class FedNovaServer(FederatedServer):
    """FedNova服务器 - 归一化平均"""

    def aggregate_fednova(self, client_updates: List[Tuple[int, Dict, int]]) -> Dict:
        """
        FedNova聚合

        Args:
            client_updates: [(client_id, state_dict, local_steps), ...]
        """
        # 计算归一化系数
        total_steps = sum(steps for _, _, steps in client_updates)

        global_dict = self.global_model.state_dict()
        aggregated = copy.deepcopy(global_dict)

        for key in aggregated.keys():
            aggregated[key] = torch.zeros_like(aggregated[key])

        # 归一化聚合
        for client_id, client_state, local_steps in client_updates:
            tau = local_steps / total_steps  # 归一化权重
            for key in aggregated.keys():
                aggregated[key] += tau * client_state[key]

        return aggregated
```

### 2.3 安全聚合

#### 2.3.1 安全多方计算（SMPC）

**秘密分享方案**：

```python
import torch
import numpy as np
from typing import List, Tuple

class SecureAggregation:
    """基于秘密分享的安全聚合"""

    def __init__(self, num_clients: int, threshold: int):
        self.n = num_clients  # 总客户端数
        self.t = threshold    # 门限值（最少需要t个客户端）

    def shamir_share(self, secret: torch.Tensor, client_id: int) -> List[Tuple[int, torch.Tensor]]:
        """
        Shamir秘密分享
        将secret分成n份，任意t份可恢复
        """
        # 生成随机多项式系数
        coeffs = [secret] + [torch.randn_like(secret) for _ in range(self.t - 1)]

        shares = []
        for i in range(1, self.n + 1):
            # 计算多项式在x=i处的值
            share = coeffs[0].clone()
            x = i
            for j in range(1, self.t):
                share += coeffs[j] * (x ** j)
            shares.append((i, share))

        return shares

    def reconstruct(self, shares: List[Tuple[int, torch.Tensor]]) -> torch.Tensor:
        """拉格朗日插值恢复秘密"""
        secret = torch.zeros_like(shares[0][1])

        for i, (xi, yi) in enumerate(shares):
            # 计算拉格朗日基多项式
            li = 1.0
            for j, (xj, _) in enumerate(shares):
                if i != j:
                    li *= (-xj) / (xi - xj)
            secret += yi * li

        return secret

    def secure_aggregate(self, client_updates: List[torch.Tensor]) -> torch.Tensor:
        """安全聚合流程"""
        n = len(client_updates)

        # 每个客户端将自己的更新分享给其他客户端
        all_shares = []
        for i, update in enumerate(client_updates):
            shares = self.shamir_share(update, i + 1)
            all_shares.append(shares)

        # 每个客户端汇总自己收到的份额
        aggregated_shares = []
        for i in range(n):
            share_sum = torch.zeros_like(all_shares[0][0][1])
            for j in range(n):
                share_sum += all_shares[j][i][1]  # 客户端i收到的来自j的份额
            aggregated_shares.append((i + 1, share_sum))

        # 恢复聚合结果
        result = self.reconstruct(aggregated_shares[:self.t])
        return result
```

#### 2.3.2 差分隐私（Differential Privacy）

**核心定义**：
一个随机机制 M 满足 (ε, δ)-差分隐私，如果对于任意相邻数据集 D, D' 和任意输出子集 S：

```
Pr[M(D) ∈ S] ≤ e^ε · Pr[M(D') ∈ S] + δ
```

**实现方案**：

```python
import torch
import numpy as np
from torch.nn.utils import clip_grad_norm_

class DPFederatedClient(FederatedClient):
    """差分隐私联邦学习客户端"""

    def __init__(self, *args,
                 noise_multiplier: float = 1.0,
                 max_grad_norm: float = 1.0,
                 delta: float = 1e-5,
                 **kwargs):
        super().__init__(*args, **kwargs)
        self.noise_multiplier = noise_multiplier
        self.max_grad_norm = max_grad_norm
        self.delta = delta
        self.epsilon = 0.0
        self.step_count = 0

    def add_noise_to_gradients(self, model: nn.Module):
        """添加高斯噪声实现差分隐私"""
        # 计算噪声标准差
        noise_std = self.noise_multiplier * self.max_grad_norm

        for param in model.parameters():
            if param.grad is not None:
                # 添加高斯噪声
                noise = torch.randn_like(param.grad) * noise_std
                param.grad += noise

    def local_train(self, global_state: Dict) -> Tuple[Dict, float]:
        self.model.load_state_dict(global_state)
        self.model.train()

        optimizer = torch.optim.SGD(self.model.parameters(), lr=self.lr)
        criterion = nn.CrossEntropyLoss()

        for epoch in range(self.local_epochs):
            for data, target in self.dataloader:
                optimizer.zero_grad()
                output = self.model(data)
                loss = criterion(output, target)
                loss.backward()

                # 1. 梯度裁剪
                clip_grad_norm_(self.model.parameters(), self.max_grad_norm)

                # 2. 添加噪声
                self.add_noise_to_gradients(self.model)

                optimizer.step()

                self.step_count += 1

        # 更新隐私预算
        self.update_privacy_budget()

        return copy.deepcopy(self.model.state_dict()), loss.item()

    def update_privacy_budget(self):
        """使用 moments accountant 计算隐私预算"""
        from opacus.accountants.utils import get_noise_multiplier

        # 简化的隐私预算计算
        q = self.batch_size / len(self.dataset)  # 采样率
        steps = self.step_count

        # 使用高级组合定理估算epsilon
        self.epsilon = self.compute_epsilon(q, steps, self.noise_multiplier, self.delta)

    def compute_epsilon(self, q, steps, noise_multiplier, delta):
        """计算隐私预算（简化版）"""
        if noise_multiplier == 0:
            return float('inf')

        # 使用RDP (Rényi Differential Privacy) 转换
        alpha = 1 + 2 * np.log(1 / delta) / noise_multiplier ** 2
        rdp = alpha * steps * q**2 / (2 * noise_multiplier**2)
        epsilon = rdp + np.log(1 / delta) / (alpha - 1)

        return epsilon


# 使用Opacus库简化实现
from opacus import PrivacyEngine

class OpacusFederatedClient(FederatedClient):
    """使用Opacus的差分隐私实现"""

    def __init__(self, *args,
                 target_epsilon: float = 3.0,
                 target_delta: float = 1e-5,
                 max_grad_norm: float = 1.0,
                 **kwargs):

        criterion = nn.CrossEntropyLoss()

            for data, target in dataloader:
                optimizer.zero_grad()

                loss = criterion(output, target)
                loss.backward()
                optimizer.step()

        # 获取隐私预算
        epsilon = privacy_engine.get_epsilon(self.target_delta)
```

#### 2.3.3 同态加密

```python
# 使用TenSEAL进行同态加密聚合
import tenseal as ts

class HEFederatedServer(FederatedServer):
    """同态加密联邦学习服务器"""

    def __init__(self, model: nn.Module):
        super().__init__(model)
        # 创建CKKS同态加密上下文
        self.context = ts.context(
            ts.SCHEME_TYPE.CKKS,
            poly_modulus_degree=8192,
            coeff_mod_bit_sizes=[60, 40, 40, 60]
        )
        self.context.global_scale = 2**40
        self.context.generate_galois_keys()

    def get_public_context(self) -> ts.Context:
        """获取公钥上下文（分发给客户端）"""
        return ts.context_from(self.context.serialize(save_secret_key=False))

    def aggregate_encrypted(self, encrypted_updates: List[ts.CKKSVector]) -> ts.CKKSVector:
        """聚合加密后的模型更新"""
        # 同态加法聚合
        result = encrypted_updates[0]
        for update in encrypted_updates[1:]:
            result = result + update

        # 平均（乘以明文分数）
        weight = 1.0 / len(encrypted_updates)
        result = result * weight

        return result


class HEFederatedClient:
    """同态加密客户端"""

    def __init__(self, client_id: int, public_context: ts.Context):
        self.client_id = client_id
        self.public_context = public_context

    def encrypt_update(self, model_update: torch.Tensor) -> ts.CKKSVector:
        """加密模型更新"""
        # 展平张量
        flat_update = model_update.flatten().tolist()

        # CKKS加密（支持浮点数）
        encrypted = ts.ckks_vector(self.public_context, flat_update)
        return encrypted

    def decrypt_update(self, encrypted_update: ts.CKKSVector,
                      original_shape: tuple) -> torch.Tensor:
        """解密模型更新"""
        decrypted = encrypted_update.decrypt()
        return torch.tensor(decrypted).view(original_shape)
```

   ```python
       """
       创建Non-IID数据分布（基于标签分片）
       """
       # 按标签排序
       sorted_indices = np.argsort(dataset.targets)

       # 分成num_clients * shards_per_client个分片
       num_shards = num_clients * shards_per_client
       shard_size = len(dataset) // num_shards

       shards = []
       for i in range(num_shards):
           start = i * shard_size
           end = start + shard_size
           shards.append(sorted_indices[start:end])

       # 每个客户端分配shards_per_client个分片
       client_datasets = []
       for i in range(num_clients):

           client_indices = np.concatenate(client_shards)
           client_datasets.append(Subset(dataset, client_indices))
   ```

   ```python
   class FairAggregation:
       """公平性聚合算法 q-FFL"""

       def aggregate_qffl(self, client_updates, client_losses, q=0.5):
           """
           q-FFL聚合：关注表现差的客户端

           Args:
               q: 公平性参数，q越大越关注差客户端
           """
           # 基于损失计算权重
           weights = []
           for loss in client_losses:
               weight = (loss ** (q + 1))
               weights.append(weight)

           # 归一化
           total = sum(weights)
           weights = [w / total for w in weights]

           # 加权聚合
           aggregated = sum(w * u for w, u in zip(weights, client_updates))
           return aggregated
   ```

```python
class ByzantineRobustAggregation:
    """拜占庭容错聚合"""

    def trimmed_mean(self, updates, trim_ratio=0.1):
        """截断均值"""
        n = len(updates)
        trim_count = int(n * trim_ratio)

        sorted_updates = sorted(updates)
        trimmed = sorted_updates[trim_count:n-trim_count]
        return np.mean(trimmed)

    def krum(self, updates, f):
        """Krum算法：选择与其他更新最接近的更新"""
        n = len(updates)
        scores = []

        for i in range(n):
            distances = []
            for j in range(n):
                if i != j:
                    dist = np.linalg.norm(updates[i] - updates[j])
                    distances.append(dist)

            distances.sort()
            score = sum(distances[:n-f-2])
            scores.append(score)

        best_idx = np.argmin(scores)
        return updates[best_idx]
```

## 五、形式化分析

### 5.1 收敛性分析

**FedAvg收敛定理**：
在满足以下条件时：

- 目标函数L是μ-强凸且L-平滑
- 随机梯度方差有界：E[||∇L_i(w) - ∇L(w)||²] ≤ σ²
- 本地数据分布差异有界

FedAvg以O(1/T)速率收敛，其中T是通信轮数。

**Non-IID情况下的收敛界限**：

```
E[L(w_T)] - L(w*) ≤ O(1/√(KT)) + O(Γ)

其中:
- K: 每轮参与的客户端数
- T: 通信轮数
- Γ: 数据异构性度量（IID时Γ=0）
```

### 5.2 隐私分析

**差分隐私组合定理**：

```
如果机制M满足(ε, δ)-DP，则k次自适应组合的隐私预算为：
ε_total ≤ kε(e^ε - 1) + ε√(2k ln(1/δ'))
δ_total = kδ + δ'
```

**联邦学习隐私-效用权衡**：

```
模型准确率 ≈ 基准准确率 - O(√d / (nε))

其中:
- d: 模型参数量
- n: 参与客户端数
- ε: 隐私预算
```
