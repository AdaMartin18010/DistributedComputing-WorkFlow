# 07-architecture - 系统架构

分布式系统架构模式与云原生技术。

## 子目录

- `cloud-computing/` - 云原生技术（Kubernetes, Docker, Serverless）
- `edge-computing/` - 边缘计算架构
- `grid-computing/` - 网格计算
- `microservices/` - 微服务架构（服务网格、熔断限流、网关）
- `p2p/` - P2P计算与DHT
- `standards/` - 架构标准规范

## 核心内容

- CNCF云原生全景图
- 微服务架构模式
- 云厂商服务对比
- 多云与混合云架构
