## йҮҮж ·зӯ–з•Ҙ

```yaml
# sampling.yaml
sampling:
  strategies:
    # й»ҳи®ӨйҮҮж ·зҺҮ
    default:
      type: probabilistic
      param: 0.1

    # жңҚеҠЎзә§еҲ«й…ҚзҪ®
    per_operation:
      service_a:
        - operation: /api/health
          type: probabilistic
          param: 0.0  # дёҚйҮҮж ·еҒҘеә·жЈҖжҹҘ
        - operation: /api/payment
          type: probabilistic
          param: 1.0  # е…ЁйҮҸйҮҮж ·ж”Ҝд»ҳжҺҘеҸЈ

      service_b:
        - operation: /api/search
          type: ratelimiting
          param: 100  # жҜҸз§’жңҖеӨҡ100дёӘ

    # иҮӘйҖӮеә”йҮҮж ·
    adaptive:
      enabled: true
      target_traces_per_second: 100
      max_operations: 2000
```

## е‘ҠиӯҰи§„еҲҷ

```yaml
# jaeger-alerts.yaml
groups:
  - name: jaeger-alerts
    rules:
      # иҝҪиёӘдёўеӨұзҺҮе‘ҠиӯҰ
      - alert: HighTraceDropRate
        expr: |
          (jaeger_collector_spans_dropped_total /
           jaeger_collector_spans_received_total) > 0.05
        for: 5m
        labels:
          severity: critical
        annotations:
          summary: "Jaeger collector dropping spans"
          description: "Drop rate is {{ $value | humanizePercentage }}"

      # еӯҳеӮЁе»¶иҝҹе‘ҠиӯҰ
      - alert: HighStorageLatency
        expr: |
          histogram_quantile(0.99,
            sum(rate(jaeger_storage_latency_bucket[5m])) by (le)
          ) > 1
        for: 10m
        labels:
          severity: warning
        annotations:
          summary: "High storage write latency"

      # жҹҘиҜўе»¶иҝҹе‘ҠиӯҰ
      - alert: HighQueryLatency
        expr: |
          histogram_quantile(0.99,
            sum(rate(jaeger_query_latency_bucket[5m])) by (le)
          ) > 5
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "High query latency"
```

## жңҖдҪіе®һи·ө

1. **йҮҮж ·зӯ–з•Ҙ**пјҡз”ҹдә§зҺҜеўғдҪҝз”ЁиҮӘйҖӮеә”йҮҮж ·пјҢе№іиЎЎеӯҳеӮЁе’Ңе®Ңж•ҙжҖ§
2. **дёҠдёӢж–Үдј ж’ӯ**пјҡзЎ®дҝқжүҖжңүжңҚеҠЎй—ҙи°ғз”ЁйғҪдј йҖ’иҝҪиёӘдёҠдёӢж–Ү
3. **ж Үзӯҫи§„иҢғ**пјҡз»ҹдёҖе‘ҪеҗҚи§„иҢғпјҢйҒҝе…Қж ҮзӯҫеҶІзӘҒ
4. **е…іиҒ”ж—Ҙеҝ—**пјҡж—Ҙеҝ—дёӯеөҢе…Ҙtrace_idпјҢе®һзҺ°иҝҪиёӘе’Ңж—Ҙеҝ—е…іиҒ”
5. **жҖ§иғҪдјҳеҢ–**пјҡеҗҲзҗҶи®ҫзҪ®зј“еҶІеҢәеӨ§е°ҸпјҢйҒҝе…ҚеҪұе“Қеә”з”ЁжҖ§иғҪ
