## 📝 使用指南

### 查找文档
1. **按主题查找**：查看核心主题域，找到对应层级
2. **按层次查找**：根据需求层次（理论/技术/实践）选择对应层级
3. **通过索引查找**：使用 `00-INDEX/` 中的索引文档

### 添加新文档
1. 确定文档所属的语义层次
2. 放置在对应的层级目录中
3. 更新 `00-INDEX/README.md`
4. 更新本文档

### 归档文档
1. 确定文档是否需要归档
2. 移动到 `archive/` 对应目录
3. 更新 `archive/归档文件清单.md`
4. 在原始位置保留引用链接（如需要）
