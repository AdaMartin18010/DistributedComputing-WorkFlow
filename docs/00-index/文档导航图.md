## 六、学习路径建议

### 初学者路径（1-2周）

1. 阅读 [分布式系统架构设计指南](13-practice/分布式系统架构设计指南.md) 了解整体架构
2. 学习 [CAP定理专题文档](02-THEORY/distributed-systems/CAP定理专题文档.md) 理解分布式系统的基本约束

4. 学习 [一致性模型专题文档](02-THEORY/distributed-systems/一致性模型专题文档.md) 理解一致性保证
5. 阅读 [TiDB架构深度分析](05-storage/newsql/TiDB架构深度分析.md) 或 [Redis深度分析](05-storage/nosql/Redis深度分析.md) 了解实际系统

1. 深入学习 [Paxos算法专题文档](02-THEORY/distributed-systems/Paxos算法专题文档.md)
