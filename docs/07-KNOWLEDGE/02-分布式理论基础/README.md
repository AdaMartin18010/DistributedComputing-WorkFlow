# 分布式理论基础

本目录包含分布式系统核心理论文档，涵盖分布式计算的基本原理、共识机制、一致性模型和时间排序等关键概念。

## 文档列表

| 文档 | 内容概要 |
|-----|---------|
| [CAP定理](CAP定理.md) | 分布式系统的三选二约束：一致性、可用性、分区容错性的权衡，PACELC扩展 |
| [一致性模型](一致性模型.md) | 从强一致性到最终一致性的各种一致性保证级别 |
| [共识算法](共识算法.md) | Paxos、Raft、Zab、PBFT等分布式一致性协议的原理与实现 |
| [时钟与排序](时钟与排序.md) | 分布式事件排序机制：逻辑时钟、向量时钟、happens-before关系 |
| [FLP不可能性](FLP不可能性.md) | 异步系统共识的理论限制及绕过方法 |

## 学习路径

### 入门路径

1. **CAP定理** → 理解分布式系统的基本权衡
2. **一致性模型** → 了解各种一致性级别
3. **时钟与排序** → 掌握分布式事件排序

### 进阶路径

1. **共识算法** → 深入理解分布式一致性实现
2. **FLP不可能性** → 理解分布式系统的理论限制
