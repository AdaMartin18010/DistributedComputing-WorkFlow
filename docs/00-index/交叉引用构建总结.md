## 二、知识网络结构

### 2.1 主题域分类

```
分布式计算与工作流知识体系
├── 理论基础
│   ├── 分布式系统理论（CAP、FLP、一致性模型）
│   ├── 共识算法（Paxos、Raft、拜占庭容错）
│   ├── 工作流理论（工作流网、工作流模式、Saga）
│   └── 形式化验证（TLA+、Petri网、CTL/LTL）
├── 技术实现
│   ├── 存储系统（NewSQL、NoSQL、DFS）
│   ├── 通信系统（消息队列、RPC、协议）
│   └── 计算系统（批处理、流处理、ML）
├── 架构设计
│   ├── 微服务架构
│   ├── 云原生架构
│   └── 边缘计算
├── 事务与一致性
│   ├── 分布式事务（2PC、3PC、TCC、Saga）
│   └── 并发控制（MVCC）
└── 可靠性与性能
    ├── 故障检测与恢复
    ├── 性能优化
    └── 可观测性
```

### 2.2 核心知识依赖链

```
理论基础链：
CAP定理 → 一致性模型 → 共识算法 → 存储系统实现

事务处理链：
一致性模型 → 分布式事务协议 → 工作流引擎 → 应用实践

微服务链：
微服务架构 → 服务治理 → 分布式事务 → 消息系统
```
