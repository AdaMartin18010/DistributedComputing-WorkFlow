# 对称加密与非对称加密

## 概述

加密是分布式系统安全的基础技术，用于保护数据的机密性和完整性。现代加密算法主要分为两大类：对称加密和非对称加密。理解它们的工作原理、适用场景以及如何协同工作，对于构建安全的分布式系统至关重要。

---

### 1.2 密码学目标

| 目标 | 描述 | 实现手段 |
|-----|------|---------|
| **机密性** (Confidentiality) | 防止未授权访问数据 | 加密算法 |
| **完整性** (Integrity) | 检测数据是否被篡改 | 消息认证码(MAC)、哈希 |
| **认证** (Authentication) | 验证数据来源 | 数字签名、MAC |
| **不可否认性** (Non-repudiation) | 防止否认行为 | 数字签名 |

## 二、对称加密

### 2.1 对称加密原理

```
对称加密特点：加密和解密使用相同密钥

┌─────────────┐                  ┌─────────────┐
│   发送方    │                  │   接收方    │
│  (Alice)    │                  │   (Bob)     │
└──────┬──────┘                  └──────┬──────┘
       │                                │
       │        共享密钥 (K)             │
       │◄───────────────────────────────│
       │    （通过安全通道预先分发）       │
       │                                │
明文(P) │                                │
  │    │                                │
  ▼    ▼                                │
┌─────────┐                             │
│ P XOR K │ ──────密文(C)──────────────▶│
│ (或其他 │                             │
│ 算法)   │                             ▼
└─────────┘                       ┌─────────┐
                                  │ C XOR K │
                                  │ (解密)  │
                                  └────┬────┘
                                       │
                                       ▼
                                  明文(P)

核心挑战：如何安全地共享密钥？
```

### 2.2 对称加密算法分类

#### 2.2.1 流加密 (Stream Cipher)

```
流加密工作原理：

明文流:    P1    P2    P3    P4    P5    ...
           │     │     │     │     │
           ▼     ▼     ▼     ▼     ▼
密钥流:    K1    K2    K3    K4    K5    ...  ← 由密钥流生成器产生
           │     │     │     │     │
           ▼     ▼     ▼     ▼     ▼
        ┌────┐┌────┐┌────┐┌────┐┌────┐
        │XOR ││XOR ││XOR ││XOR ││XOR │
        └────┘└────┘└────┘└────┘└────┘
           │     │     │     │     │
           ▼     ▼     ▼     ▼     ▼
密文流:    C1    C2    C3    C4    C5    ...

特点：
• 逐字节/位加密，速度快
• 适合实时数据流（如视频通话）
• 密钥流不能重复使用！
```

**Python ChaCha20流加密示例**：

```python
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
from cryptography.hazmat.backends import default_backend
import os
import base64

class StreamEncryption:
    """
    ChaCha20流加密实现
    ChaCha20是现代的流加密算法，被TLS 1.3采用
    """

    def __init__(self):
        self.key_size = 32  # 256位密钥
        self.nonce_size = 16  # 128位nonce

    def generate_key(self) -> bytes:
        """生成随机密钥"""
        return os.urandom(self.key_size)

    def encrypt(self, plaintext: bytes, key: bytes) -> dict:
        """
        使用ChaCha20加密数据

        Args:
            plaintext: 待加密数据
            key: 32字节密钥

        Returns:
            dict包含密文和nonce（nonce可以公开）
        """
        # 生成随机nonce（每个消息必须唯一）
        nonce = os.urandom(self.nonce_size)

        # 创建ChaCha20加密器
        algorithm = algorithms.ChaCha20(key, nonce)
        cipher = Cipher(algorithm, mode=None, backend=default_backend())
        encryptor = cipher.encryptor()

        # 加密
        ciphertext = encryptor.update(plaintext) + encryptor.finalize()

        return {
            'ciphertext': base64.b64encode(ciphertext).decode(),
            'nonce': base64.b64encode(nonce).decode()
        }

    def decrypt(self, ciphertext_b64: str, nonce_b64: str,
                key: bytes) -> bytes:
        """
        解密ChaCha20密文
        """
        ciphertext = base64.b64decode(ciphertext_b64)
        nonce = base64.b64decode(nonce_b64)

        algorithm = algorithms.ChaCha20(key, nonce)
        cipher = Cipher(algorithm, mode=None, backend=default_backend())
        decryptor = cipher.decryptor()

        return decryptor.update(ciphertext) + decryptor.finalize()


# 使用示例
if __name__ == "__main__":
    crypto = StreamEncryption()

    # 生成密钥（实际应安全存储）
    key = crypto.generate_key()
    print(f"Generated Key: {base64.b64encode(key).decode()[:20]}...")

    # 加密消息
    message = b"Hello, this is a secret message for distributed system!"
    encrypted = crypto.encrypt(message, key)
    print(f"\nEncrypted (base64): {encrypted['ciphertext'][:50]}...")

    # 解密消息
    decrypted = crypto.decrypt(
        encrypted['ciphertext'],
        encrypted['nonce'],
        key
    )
    print(f"Decrypted: {decrypted.decode()}")
```

**AES-GCM完整实现**：

```python
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
import os
import base64
import json

class AESGCMCipher:
    """
    AES-GCM认证加密实现
    提供机密性和完整性保护
    """

    def __init__(self):
        self.key_size = 32  # AES-256
        self.nonce_size = 12  # GCM推荐96位nonce

    def derive_key(self, password: str, salt: bytes) -> bytes:
        """
        从密码派生密钥（PBKDF2）

        Args:
            password: 用户密码
            salt: 随机盐值（应存储）
        """
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=self.key_size,
            salt=salt,
            iterations=480000,  # OWASP推荐值
        )
        return kdf.derive(password.encode())

    def encrypt(self, plaintext: bytes, key: bytes,
                associated_data: bytes = None) -> dict:
        """
        AES-GCM加密

        Args:
            plaintext: 待加密数据
            key: 32字节密钥
            associated_data: 关联数据（认证但不加密，如消息头）

        Returns:
            包含密文、nonce和认证标签的字典
        """
        aesgcm = AESGCM(key)
        nonce = os.urandom(self.nonce_size)

        # 加密并生成认证标签
        ciphertext_with_tag = aesgcm.encrypt(
            nonce,
            plaintext,
            associated_data
        )

        # AESGCM.encrypt返回 ciphertext + 16-byte tag
        ciphertext = ciphertext_with_tag[:-16]
        tag = ciphertext_with_tag[-16:]

        return {
            'ciphertext': base64.b64encode(ciphertext).decode(),
            'nonce': base64.b64encode(nonce).decode(),
            'tag': base64.b64encode(tag).decode(),
            'associated_data': base64.b64encode(associated_data).decode() if associated_data else None
        }

    def decrypt(self, encrypted_data: dict, key: bytes) -> bytes:
        """
        AES-GCM解密

        如果数据被篡改，将抛出InvalidTag异常
        """
        aesgcm = AESGCM(key)

        ciphertext = base64.b64decode(encrypted_data['ciphertext'])
        nonce = base64.b64decode(encrypted_data['nonce'])
        tag = base64.b64decode(encrypted_data['tag'])

        associated_data = None
        if encrypted_data.get('associated_data'):
            associated_data = base64.b64decode(encrypted_data['associated_data'])

        # 重新组合 ciphertext + tag
        ciphertext_with_tag = ciphertext + tag

        return aesgcm.decrypt(nonce, ciphertext_with_tag, associated_data)

    def encrypt_password_based(self, plaintext: bytes,
                               password: str) -> dict:
        """
        基于密码的加密（自动处理盐值）
        """
        salt = os.urandom(16)
        key = self.derive_key(password, salt)

        result = self.encrypt(plaintext, key)
        result['salt'] = base64.b64encode(salt).decode()
        del result['associated_data']  # 密码加密不使用关联数据

        return result

    def decrypt_password_based(self, encrypted_data: dict,
                               password: str) -> bytes:
        """基于密码的解密"""
        salt = base64.b64decode(encrypted_data['salt'])
        key = self.derive_key(password, salt)

        return self.decrypt(encrypted_data, key)


# 使用示例：加密敏感配置
if __name__ == "__main__":
    cipher = AESGCMCipher()

    # 场景1：使用随机密钥加密服务间通信
    key = os.urandom(32)

    # 加密带有上下文的API消息
    message = json.dumps({
        'user_id': '12345',
        'action': 'transfer',
        'amount': 1000.00
    }).encode()

    # 关联数据：用于认证但不加密（如消息类型、版本等）
    associated = b'api:v1:payment'

    encrypted = cipher.encrypt(message, key, associated)
    print("Encrypted message:")
    print(json.dumps(encrypted, indent=2))

    # 解密
    decrypted = cipher.decrypt(encrypted, key)
    print(f"\nDecrypted: {decrypted.decode()}")

    # 验证完整性：篡改密文
    print("\n--- 完整性验证 ---")
    tampered = encrypted.copy()
    tampered['ciphertext'] = base64.b64encode(
        base64.b64decode(tampered['ciphertext'])[:-1] + b'X'
    ).decode()

    try:
        cipher.decrypt(tampered, key)
    except Exception as e:
        print(f"Tampering detected! Error: {e}")
```

### 2.3 对称加密算法对比

| 算法 | 类型 | 密钥长度 | 块大小 | 推荐场景 |
|-----|------|---------|--------|---------|
| **AES-256-GCM** | 块加密 | 256位 | 128位 | 通用加密、TLS |
| **ChaCha20-Poly1305** | 流加密 | 256位 | - | 移动设备、低功耗设备 |

| **3DES** | 块加密 | 168位 | 64位 | 遗留系统（已不推荐） |

## 三、非对称加密

### 3.1 非对称加密原理

```
非对称加密：使用密钥对（公钥 + 私钥）

密钥生成：
┌─────────────────────────────────────────────┐
│              密钥对生成算法                   │
│                                             │
│  输入: 随机数（熵源）                         │
│       │                                     │
│       ▼                                     │
│  ┌─────────┐                                │
│  │ 数学运算 │  (大素数分解、椭圆曲线等)        │
│  │ (RSA/ECC)│                                │
│  └────┬────┘                                │
│       │                                     │
│       ├──▶ 公钥 (Public Key) - 可以公开      │
│       │                                     │
│       └──▶ 私钥 (Private Key) - 严格保密     │
└─────────────────────────────────────────────┘

加密流程：
┌─────────────┐                  ┌─────────────┐
│   Alice     │                  │    Bob      │
│  (发送方)   │                  │  (接收方)   │
└──────┬──────┘                  └──────┬──────┘
       │                                │
       │        Bob的公钥               │
       │◄───────────────────────────────│
       │        (从公开渠道获取)          │
       │                                │
明文(P)│                                │
  │    │                                │
  ▼    ▼                                │
┌─────────┐                             │
│  公钥加密 │                             │
│Encrypt(P,│ ──────密文(C)──────────────▶│
│PubKey_B)│                             │
└─────────┘                             ▼
                                  ┌─────────┐
                                  │  私钥解密 │
                                  │Decrypt(C,│
                                  │PriKey_B)│
                                  └────┬────┘
                                       │
                                       ▼
                                  明文(P)

关键特性：
• 公钥加密的数据只能用对应的私钥解密
• 私钥加密的数据（数字签名）可用公钥验证
• 无法从公钥推导出私钥
```

### 3.2 RSA算法详解

```
RSA算法基于大整数分解的数学难题

密钥生成：
1. 选择两个大素数 p 和 q
2. 计算 n = p × q （模数，通常2048位或更大）
3. 计算 φ(n) = (p-1)(q-1)
4. 选择公钥指数 e（通常为65537）
5. 计算私钥 d ≡ e^(-1) mod φ(n)

公钥: (e, n)
私钥: (d, n)

加密: C = M^e mod n
解密: M = C^d mod n

安全性：从n分解出p和q在计算上不可行
```

**RSA完整实现**：

```python
from cryptography.hazmat.primitives.asymmetric import rsa, padding
from cryptography.hazmat.primitives import hashes, serialization
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
from cryptography.hazmat.backends import default_backend
import os
import base64

class RSAEncryption:
    """
    RSA非对称加密实现
    适用于密钥交换、数字签名
    """

    def __init__(self, key_size: int = 2048):
        self.key_size = key_size

    def generate_keypair(self) -> tuple:
        """
        生成RSA密钥对

        Returns:
            (private_key, public_key) - 密钥对象
        """
        private_key = rsa.generate_private_key(
            public_exponent=65537,
            key_size=self.key_size,
            backend=default_backend()
        )
        public_key = private_key.public_key()

        return private_key, public_key

    def export_private_key(self, private_key, password: str = None) -> bytes:
        """
        导出私钥（PEM格式）

        Args:
            password: 如果提供，使用密码加密私钥
        """
        encryption_algorithm = serialization.NoEncryption()
        if password:
            encryption_algorithm = serialization.BestAvailableEncryption(
                password.encode()
            )

        return private_key.private_bytes(
            encoding=serialization.Encoding.PEM,
            format=serialization.PrivateFormat.PKCS8,
            encryption_algorithm=encryption_algorithm
        )

    def export_public_key(self, public_key) -> bytes:
        """导出公钥（PEM格式）"""
        return public_key.public_bytes(
            encoding=serialization.Encoding.PEM,
            format=serialization.PublicFormat.SubjectPublicKeyInfo
        )

    def load_private_key(self, pem_data: bytes, password: str = None):
        """从PEM加载私钥"""
        return serialization.load_pem_private_key(
            pem_data,
            password=password.encode() if password else None,
            backend=default_backend()
        )

    def load_public_key(self, pem_data: bytes):
        """从PEM加载公钥"""
        return serialization.load_pem_public_key(
            pem_data,
            backend=default_backend()
        )

    def encrypt(self, plaintext: bytes, public_key) -> bytes:
        """
        RSA加密（OAEP填充）

        注意：RSA不适合直接加密大量数据
        最大加密长度 = key_size/8 - 2*hash_size - 2
        对于2048位RSA和SHA256，最大为190字节
        """
        return public_key.encrypt(
            plaintext,
            padding.OAEP(
                mgf=padding.MGF1(algorithm=hashes.SHA256()),
                algorithm=hashes.SHA256(),
                label=None
            )
        )

    def decrypt(self, ciphertext: bytes, private_key) -> bytes:
        """RSA解密"""
        return private_key.decrypt(
            ciphertext,
            padding.OAEP(
                mgf=padding.MGF1(algorithm=hashes.SHA256()),
                algorithm=hashes.SHA256(),
                label=None
            )
        )

    def sign(self, message: bytes, private_key) -> bytes:
        """
        RSA数字签名（PSS填充）
        """
        return private_key.sign(
            message,
            padding.PSS(
                mgf=padding.MGF1(hashes.SHA256()),
                salt_length=padding.PSS.MAX_LENGTH
            ),
            hashes.SHA256()
        )

    def verify(self, message: bytes, signature: bytes, public_key) -> bool:
        """
        验证RSA签名

        Returns:
            验证成功返回True，失败返回False
        """
        try:
            public_key.verify(
                signature,
                message,
                padding.PSS(
                    mgf=padding.MGF1(hashes.SHA256()),
                    salt_length=padding.PSS.MAX_LENGTH
                ),
                hashes.SHA256()
            )
            return True
        except Exception:
            return False


# 混合加密：RSA + AES
class HybridEncryption:
    """
    混合加密方案：RSA加密AES密钥，AES加密数据
    结合两种算法的优势
    """

    def __init__(self):
        self.rsa = RSAEncryption()

    def encrypt(self, plaintext: bytes, public_key) -> dict:
        """
        使用接收方的公钥加密数据

        流程：
        1. 生成随机的AES密钥
        2. 用AES加密数据
        3. 用RSA加密AES密钥
        4. 返回加密后的数据和密钥
        """
        # 生成随机AES-256密钥
        aes_key = os.urandom(32)

        # AES-GCM加密数据
        from cryptography.hazmat.primitives.ciphers.aead import AESGCM
        aesgcm = AESGCM(aes_key)
        nonce = os.urandom(12)
        ciphertext = aesgcm.encrypt(nonce, plaintext, None)

        # RSA加密AES密钥
        encrypted_key = self.rsa.encrypt(aes_key, public_key)

        return {
            'encrypted_key': base64.b64encode(encrypted_key).decode(),
            'nonce': base64.b64encode(nonce).decode(),
            'ciphertext': base64.b64encode(ciphertext).decode()
        }

    def decrypt(self, encrypted_package: dict, private_key) -> bytes:
        """
        使用接收方的私钥解密数据
        """
        # RSA解密AES密钥
        encrypted_key = base64.b64decode(encrypted_package['encrypted_key'])
        aes_key = self.rsa.decrypt(encrypted_key, private_key)

        # AES解密数据
        from cryptography.hazmat.primitives.ciphers.aead import AESGCM
        aesgcm = AESGCM(aes_key)
        nonce = base64.b64decode(encrypted_package['nonce'])
        ciphertext = base64.b64decode(encrypted_package['ciphertext'])

        return aesgcm.decrypt(nonce, ciphertext, None)


# 使用示例
```

## 四、密钥交换协议

### 4.1 Diffie-Hellman密钥交换

```
Diffie-Hellman密钥交换原理：

不需要共享密钥，双方协商出共享密钥

公共参数（公开）：
  g = 生成元
  p = 大素数

Alice                          Bob
  │                              │
  │  生成私钥 a (随机)            │
  │  计算 A = g^a mod p          │
  │                              │  生成私钥 b (随机)
  │                              │  计算 B = g^b mod p
  │                              │
  │  发送 A ────────────────────▶│
  │◀─────────────────────────────│  发送 B
  │                              │
  │  计算 s = B^a mod p          │  计算 s = A^b mod p
  │         = (g^b)^a mod p      │         = (g^a)^b mod p
  │         = g^(ab) mod p       │         = g^(ab) mod p
  │                              │
  ▼                              ▼
共享密钥: s = g^(ab) mod p (双方计算结果相同)

攻击者即使截获A、B、g、p，也无法计算出s（离散对数难题）
```

**Python ECDH实现**：

```python
from cryptography.hazmat.primitives.asymmetric import ec
from cryptography.hazmat.primitives import hashes, serialization
from cryptography.hazmat.primitives.kdf.hkdf import HKDF
import base64

class ECDHKeyExchange:
    """
    椭圆曲线Diffie-Hellman密钥交换 (ECDH)
    用于安全地协商共享密钥
    """

    def __init__(self, curve=ec.SECP256R1()):
        self.curve = curve
        self.private_key = None
        self.public_key = None

    def generate_keypair(self):
        """生成EC密钥对"""
        self.private_key = ec.generate_private_key(self.curve)
        self.public_key = self.private_key.public_key()
        return self.public_key

    def get_public_key_bytes(self) -> bytes:
        """获取公钥字节表示（用于传输）"""
        return self.public_key.public_bytes(
            encoding=serialization.Encoding.X962,
            format=serialization.PublicFormat.UncompressedPoint
        )

    def derive_shared_key(self, peer_public_key_bytes: bytes) -> bytes:
        """
        使用对方公钥派生共享密钥

        Returns:
            32字节共享密钥（可用于AES-256）
        """
        # 加载对方公钥
        peer_public_key = ec.EllipticCurvePublicKey.from_encoded_point(
            self.curve,
            peer_public_key_bytes
        )

        # ECDH密钥交换
        shared_secret = self.private_key.exchange(ec.ECDH(), peer_public_key)

        # 使用HKDF派生密钥
        derived_key = HKDF(
            algorithm=hashes.SHA256(),
            length=32,
            salt=None,
            info=b'handshake data'
        ).derive(shared_secret)

        return derived_key


# 完整密钥交换示例
if __name__ == "__main__":
    print("=== ECDH Key Exchange ===\n")

    # Alice生成密钥对
    alice = ECDHKeyExchange()
    alice_public = alice.generate_keypair()
    alice_public_bytes = alice.get_public_key_bytes()

    # Bob生成密钥对
    bob = ECDHKeyExchange()
    bob_public = bob.generate_keypair()
    bob_public_bytes = bob.get_public_key_bytes()

    print(f"Alice public key length: {len(alice_public_bytes)} bytes")
    print(f"Bob public key length: {len(bob_public_bytes)} bytes")

    # 交换公钥并派生共享密钥
    alice_shared = alice.derive_shared_key(bob_public_bytes)
    bob_shared = bob.derive_shared_key(alice_public_bytes)

    print(f"\nAlice derived key: {base64.b64encode(alice_shared).decode()}")
    print(f"Bob derived key:   {base64.b64encode(bob_shared).decode()}")
    print(f"\nKeys match: {alice_shared == bob_shared}")

    # 现在可以使用共享密钥进行对称加密
    from cryptography.hazmat.primitives.ciphers.aead import AESGCM

    aesgcm = AESGCM(alice_shared)
    nonce = b'unique_nonce_12'  # 12字节
    message = b"Secret message between Alice and Bob"

    ciphertext = aesgcm.encrypt(nonce, message, None)
    print(f"\nEncrypted message: {base64.b64encode(ciphertext).decode()[:50]}...")

    # Bob解密
    aesgcm_bob = AESGCM(bob_shared)
    decrypted = aesgcm_bob.decrypt(nonce, ciphertext, None)
    print(f"Bob decrypted: {decrypted.decode()}")
```

## 六、最佳实践

### 6.1 密钥管理原则

1. **密钥生成**：使用加密安全的随机数生成器（CSPRNG）
2. **密钥存储**：使用HSM或密钥管理服务（KMS）
3. **密钥传输**：使用密钥封装（如RSA-KEM）或DH密钥交换
4. **密钥轮换**：定期更换密钥，支持密钥版本管理
5. **密钥销毁**：安全擦除不再使用的密钥

### 6.2 实施检查清单

- [ ] 使用AES-256-GCM或ChaCha20-Poly1305进行数据加密
- [ ] 使用RSA-2048+或ECC P-256+进行密钥交换和签名
- [ ] 每个加密操作使用唯一的IV/Nonce
- [ ] 验证密文的完整性（MAC/认证标签）
- [ ] 敏感数据加密后存储
- [ ] 传输数据使用TLS 1.3
- [ ] 密码使用Argon2id进行哈希存储
