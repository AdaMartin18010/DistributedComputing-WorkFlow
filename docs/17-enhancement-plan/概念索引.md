## 🔗 概念关系索引

### 依赖关系

- **TLA+** → 状态、动作、行为
- **CTL** → Kripke结构、路径
- **LTL** → 路径
- **Petri网** → 位置、转换、标记
- **工作流网** → Petri网
- **CAP定理** → 一致性、可用性、分区容错性
- **FLP不可能定理** → 异步系统、共识问题
- **一致性模型** → 操作历史
- **Paxos算法** → 提议者、接受者、学习者
- **Raft算法** → 节点角色、任期、日志条目
- **Saga模式** → 补偿操作、最终一致性

### 层次关系

- **时序逻辑** → TLA+、CTL、LTL
- **Petri网** → 工作流网
- **一致性模型** → 线性一致性、顺序一致性、因果一致性、最终一致性
- **共识算法** → Paxos算法、Raft算法

### 等价关系

- **TLA+中的行为** ≈ **CTL/LTL中的路径**
- **Petri网中的标记** ≈ **工作流网中的状态**

### 蕴含关系

- **线性一致性** → **顺序一致性**
- **顺序一致性** → **因果一致性**
- **因果一致性** → **最终一致性**

### 互斥关系

- **CAP定理中的一致性** ↔ **可用性**（在网络分区时）
