## 🔧 实施方法

### 方法1：可折叠图表说明

**适用场景**：复杂的Mermaid图表

**实施步骤**：

1. 在图表前添加`<details>`标签
2. 添加图表说明内容
3. 使用`<summary>`标签作为标题

**示例**：

```markdown
      <details>
      <summary><strong>📊 工作流执行流程图（点击展开说明）</strong></summary>

      **图表用途**：展示Temporal工作流的执行流程

      **关键节点**：
      - Start：工作流开始
      - Activity：活动执行
      - End：工作流结束

      </details>

      ```mermaid
      graph TD
         A[Start] --> B[Activity]
         B --> C[End]
      ```

```

---

### 方法2：代码块折叠

**适用场景**：长代码示例

**实施步骤**：

1. 使用`<details>`标签包装代码块
2. 添加代码说明
3. 保持代码格式

**示例**：

```markdown
<details>
<summary><strong>💻 完整工作流定义（点击展开）</strong></summary>

```python
@workflow.defn
class OrderWorkflow:
    @workflow.run
    async def execute(self, order_id: str) -> str:
        return "Order processed"
```

</details>
```

---
