## 📝 图表优化标准

### 思维导图优化标准

1. **层次结构**：确保层次结构清晰，根节点、一级节点、二级节点、三级节点层次分明
2. **内容完整性**：确保核心概念、形式化定义、性质与定理、应用场景等关键内容完整
3. **节点命名**：节点命名准确、简洁、有描述性
4. **连接关系**：节点之间的连接关系准确、有意义

### 决策树优化标准

1. **逻辑流程**：决策流程逻辑清晰，从起始节点到终点节点的路径明确
2. **决策节点**：决策节点的问题准确、选项完整
3. **结果节点**：结果节点清晰、准确
4. **样式统一**：使用统一的样式（颜色、形状等）

### 对比矩阵优化标准

1. **数据准确性**：矩阵中的数据准确、有依据
2. **格式一致性**：矩阵格式统一、易读
3. **评分依据**：评分有明确的依据说明
4. **完整性**：对比维度完整、覆盖全面

### 关系图优化标准

1. **节点关系**：节点之间的关系准确、有意义
2. **连接准确性**：连接线准确反映节点关系
3. **分组清晰**：相关节点分组清晰
4. **样式统一**：使用统一的样式

### 流程图优化标准

1. **流程逻辑**：流程逻辑清晰、完整
2. **节点完整性**：流程节点完整、无遗漏
3. **连接准确性**：节点之间的连接准确
4. **样式统一**：使用统一的样式
