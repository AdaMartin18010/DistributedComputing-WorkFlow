# 差分隐私

## 概述

差分隐私（Differential Privacy）是一种保护数据隐私的严格数学框架。它通过在查询结果中添加精心设计的噪声，确保单个数据记录的存在与否不会显著影响输出结果，从而保护个体隐私。在分布式系统中，差分隐私被广泛应用于数据分析、机器学习等领域。

---

## 一、差分隐私基础

### 1.1 核心概念

```
差分隐私直观理解：

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  场景：医院发布疾病统计数据                                   │
│                                                             │
│  传统方法：直接发布统计结果                                    │
│  ┌─────────────┐                                            │
│  │ 数据库      │──▶ 查询：糖尿病患者数量 = 1000             │
│  │ (含个人记录) │──▶ 发布：1000                              │
│  └─────────────┘                                            │
│                                                             │
│  问题：攻击者知道999人，通过差分推断第1000人是否患病          │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  差分隐私方法：添加噪声                                       │
│  ┌─────────────┐                                            │
│  │ 数据库      │──▶ 查询：糖尿病患者数量                     │
│  │ (含个人记录) │──▶ 添加噪声：Laplace(1000, ε)            │
│  └─────────────┘──▶ 发布：1002 ± 5 （不确定的真实值）        │
│                                                             │
│  效果：                                                       │
│  • 第1000人是否存在对结果影响很小                             │
│  • 无法通过发布结果推断个体信息                                │
│  • 但仍能保持整体统计准确性                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 形式化定义

```
(ε, δ)-差分隐私定义：

一个随机化机制 M: D → R 满足 (ε, δ)-差分隐私，当且仅当对于
任意两个相邻数据集 D 和 D'（相差一个记录），以及任意输出子集 S ⊆ R：

    Pr[M(D) ∈ S] ≤ e^ε × Pr[M(D') ∈ S] + δ

其中：
• ε (epsilon): 隐私预算，越小隐私保护越强
• δ (delta): 失败的概率，通常要求 < 1/n²

特殊情况：
• 当 δ = 0 时，称为纯差分隐私 (ε-DP)
• ε-DP 提供最强的隐私保证

隐私预算累积：
• 多次查询会累积消耗隐私预算
• 组合定理：k 个 ε-DP 机制的组合满足 kε-DP
```

### 1.3 隐私预算

```
隐私预算 ε 的选择：

┌─────────────────────────────────────────────────────────────┐
│  ε 值范围      │ 隐私级别    │ 适用场景                     │
├─────────────────────────────────────────────────────────────┤
│  ε ≤ 0.1      │ 极强保护    │ 高度敏感数据（医疗、金融）     │
│  0.1 < ε ≤ 1  │ 强保护      │ 一般敏感数据                 │
│  1 < ε ≤ 3    │ 中等保护    │ 统计分析、聚合数据            │
```

## 二、噪声机制

### 2.1 拉普拉斯机制

```
拉普拉斯机制（Laplace Mechanism）：

适用于：数值型查询，如计数、求和、均值等

噪声分布：

    Lap(x|b) = (1/2b) × exp(-|x|/b)

其中 b = GS(f) / ε

GS(f) 是查询函数 f 的全局敏感度：
GS(f) = max |f(D) - f(D')|  对任意相邻数据集 D, D'

┌─────────────────────────────────────────────────────────────┐
│                    拉普拉斯分布 PDF                          │
│                                                             │
│      │                                                      │
│      │     ╲                                                 │
│      │      ╲                                                │
│      │       ╲                                               │
│      │        ╲                                              │
│      │         ╲                                             │
│      │          ╲                   ╱                        │
│      │           ╲                 ╱                         │
│      │            ╲               ╱                          │
│      │             ╲             ╱                           │
│      │              ╲           ╱                            │
│      │               ╲         ╱                             │
│      │                ╲_______╱                              │
│      └───────────────────────────────────▶                   │
│                      x=0                                    │
│                                                             │
│  特点：                                                       │
│  • 对称分布，峰值在0                                         │
│  • 尺度参数 b 越大，噪声越大                                  │
│  • 概率密度随距离指数衰减                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 高斯机制

```
高斯机制（Gaussian Mechanism）：

适用于：(ε, δ)-差分隐私场景

噪声分布：N(0, σ²)

其中 σ = GS(f) × √(2ln(1.25/δ)) / ε

相比拉普拉斯机制：
• 提供 (ε, δ)-DP 而非纯 ε-DP
• 对于高维数据噪声更小
• 适合机器学习中梯度添加噪声

选择建议：
• 纯计数查询 → 拉普拉斯机制
• 高维向量/深度学习 → 高斯机制
```

```python
import numpy as np
from typing import Callable, Union
import math

class DifferentialPrivacyMechanism:
    """
    差分隐私机制实现
    """
    
    def __init__(self, epsilon: float, delta: float = 0.0):
        """
        Args:
            epsilon: 隐私预算
            delta: 近似差分隐私参数，0表示纯ε-DP
        """
        self.epsilon = epsilon
        self.delta = delta
    
    def laplace_mechanism(self, 
                         true_value: Union[float, np.ndarray],
                         sensitivity: float) -> Union[float, np.ndarray]:
        """
        拉普拉斯机制
        
        Args:
            true_value: 真实查询结果
            sensitivity: 查询的全局敏感度
            
        Returns:
            添加噪声后的结果
        """
        # 拉普拉斯分布的尺度参数
        scale = sensitivity / self.epsilon
        
        # 生成拉普拉斯噪声
        # Laplace(0, b) = b * (U1 - U2) where U1, U2 ~ Exp(1)
        if isinstance(true_value, np.ndarray):
            noise = np.random.laplace(0, scale, size=true_value.shape)
        else:
            noise = np.random.laplace(0, scale)
        
        return true_value + noise
    
    def gaussian_mechanism(self,
                          true_value: Union[float, np.ndarray],
                          sensitivity: float) -> Union[float, np.ndarray]:
        """
        高斯机制（(ε, δ)-差分隐私）
        """
        if self.delta == 0:
            raise ValueError("Gaussian mechanism requires delta > 0")
        
        # 计算标准差
        sigma = sensitivity * math.sqrt(2 * math.log(1.25 / self.delta)) / self.epsilon
        
        # 生成高斯噪声
        if isinstance(true_value, np.ndarray):
            noise = np.random.normal(0, sigma, size=true_value.shape)
        else:
            noise = np.random.normal(0, sigma)
        
        return true_value + noise
    
    def exponential_mechanism(self,
                             candidates: list,
                             utility_function: Callable,
                             sensitivity: float) -> any:
        """
        指数机制
        用于非数值型输出（如选择最优类别）
        
        Args:
            candidates: 候选结果集合
            utility_function: 效用函数 u(D, r)
            sensitivity: 效用函数的全局敏感度
            
        Returns:
            根据指数分布选择的候选
        """
        # 计算每个候选的效用分数
        scores = [utility_function(c) for c in candidates]
        
        # 计算概率权重: exp(ε * u(D,r) / (2 * Δu))
        weights = [math.exp(self.epsilon * s / (2 * sensitivity)) for s in scores]
        
        # 归一化概率
        total = sum(weights)
        probabilities = [w / total for w in weights]
        
        # 按概率采样
        return np.random.choice(candidates, p=probabilities)


# 敏感度计算工具
class SensitivityCalculator:
    """
    查询敏感度计算
    """
    
    @staticmethod
    def count_sensitivity() -> float:
        """
        计数查询的敏感度
        添加/删除一个记录最多改变计数1
        """
        return 1.0
    
    @staticmethod
    def sum_sensitivity(lower_bound: float, upper_bound: float) -> float:
        """
        求和查询的敏感度
        
        Args:
            lower_bound: 数据项最小值
            upper_bound: 数据项最大值
        """
        return abs(upper_bound - lower_bound)
    
    @staticmethod
    def mean_sensitivity(lower_bound: float, upper_bound: float, 
                        min_count: int) -> float:
        """
        均值查询的敏感度（有界数据）
        """
        return (upper_bound - lower_bound) / min_count
    
    @staticmethod
    def vector_sensitivity(norm_bound: float) -> float:
        """
        向量查询的敏感度（L2范数）
        常用于机器学习梯度
        """
        return norm_bound
```

### 3.1 隐私预算组合

```python
class PrivacyBudgetAccountant:
    """
    隐私预算会计
    跟踪和管理隐私预算消耗
    """
    
    def __init__(self, total_epsilon: float, total_delta: float):
        self.total_epsilon = total_epsilon
        self.total_delta = total_delta
        self.consumed_epsilon = 0.0
        self.consumed_delta = 0.0
        self.query_log = []
    
    def check_budget(self, epsilon: float, delta: float = 0.0) -> bool:
        """检查是否有足够预算"""
        return (self.consumed_epsilon + epsilon <= self.total_epsilon and
                self.consumed_delta + delta <= self.total_delta)
    
    def consume(self, epsilon: float, delta: float = 0.0, 
                query_info: str = "") -> bool:
        """
        消耗隐私预算
        
        Returns:
            是否成功消费
        """
        if not self.check_budget(epsilon, delta):
            return False
        
        self.consumed_epsilon += epsilon
        self.consumed_delta += delta
        self.query_log.append({
            'epsilon': epsilon,
            'delta': delta,
            'info': query_info,
            'remaining': self.get_remaining()
        })
        
        return True
    
    def get_remaining(self) -> dict:
        """获取剩余预算"""
        return {
            'epsilon': self.total_epsilon - self.consumed_epsilon,
            'delta': self.total_delta - self.consumed_delta
        }
    
    @staticmethod
    def advanced_composition(k: int, epsilon: float, 
                            delta: float) -> tuple:
        """
        高级组合定理
        k个(ε, δ)-DP机制的组合满足(ε', kδ+δ')-DP
        
        Returns:
            (组合后的epsilon, 组合后的delta)
        """
        # 简化版本，实际使用更复杂的公式
        epsilon_prime = epsilon * math.sqrt(2 * k * math.log(1 / delta))
        delta_prime = k * delta
        
        return epsilon_prime, delta_prime


# 差分隐私数据库查询
class PrivateDatabaseQuery:
    """
    差分隐私数据库查询接口
    """
    
    def __init__(self, data: np.ndarray, 
                 budget_accountant: PrivacyBudgetAccountant):
        self.data = data
        self.accountant = budget_accountant
        self.dp = DifferentialPrivacyMechanism(
            epsilon=1.0  # 每次查询的默认预算
        )
    
    def private_count(self, predicate: Callable, 
                      epsilon: float = None) -> float:
        """
        差分隐私计数查询
        
        Args:
            predicate: 过滤条件函数
            epsilon: 本次查询的隐私预算
        """
        if epsilon is None:
            epsilon = self.dp.epsilon
        
        if not self.accountant.consume(epsilon, 0.0, "count"):
            raise RuntimeError("Privacy budget exhausted")
        
        # 计算真实计数
        true_count = np.sum(predicate(self.data))
        
        # 添加噪声
        return self.dp.laplace_mechanism(
            float(true_count),
            SensitivityCalculator.count_sensitivity()
        )
    
    def private_sum(self, column_index: int,
                   bounds: tuple,
                   epsilon: float = None) -> float:
        """差分隐私求和查询"""
        if epsilon is None:
            epsilon = self.dp.epsilon
        
        if not self.accountant.consume(epsilon, 0.0, "sum"):
            raise RuntimeError("Privacy budget exhausted")
        
        # 数据裁剪到边界
        clipped_data = np.clip(self.data[:, column_index], 
                               bounds[0], bounds[1])
        
        true_sum = np.sum(clipped_data)
        
        return self.dp.laplace_mechanism(
            float(true_sum),
            SensitivityCalculator.sum_sensitivity(bounds[0], bounds[1])
        )
    
    def private_mean(self, column_index: int,
                    bounds: tuple,
                    epsilon: float = None) -> float:
        """
        差分隐私均值查询
        使用分解：mean = sum / count
        """
        if epsilon is None:
            epsilon = self.dp.epsilon
        
        # 分配预算（各一半）
        epsilon_per_query = epsilon / 2
        
        private_sum = self.private_sum(column_index, bounds, epsilon_per_query)
        private_count = self.private_count(
            lambda x: np.ones(len(x), dtype=bool),
            epsilon_per_query
        )
        
        # 避免除以零
        if private_count <= 0:
            return float('nan')
        
        return private_sum / private_count
    
    def private_histogram(self, column_index: int,
                         bins: int,
                         epsilon: float = None) -> np.ndarray:
        """
        差分隐私直方图
        """
        if epsilon is None:
            epsilon = self.dp.epsilon
        
        if not self.accountant.consume(epsilon, 0.0, "histogram"):
            raise RuntimeError("Privacy budget exhausted")
        
        # 计算直方图
        hist, _ = np.histogram(self.data[:, column_index], bins=bins)
        
        # 每个桶添加噪声（并行组合）
        # 敏感度为1，因为一个记录只影响一个桶
        epsilon_per_bin = epsilon  # 并行组合不分割预算
        
        noisy_hist = self.dp.laplace_mechanism(
            hist.astype(float),
            sensitivity=1.0
        )
        
        return noisy_hist
```

### 3.2 本地差分隐私（LDP）

```python
class LocalDifferentialPrivacy:
    """
    本地差分隐私
    数据在发送到服务器之前就添加了噪声
    更强的隐私保护，但数据效用较低
    """
    
    def __init__(self, epsilon: float):
        self.epsilon = epsilon
    
    def randomized_response(self, true_value: bool) -> bool:
        """
        随机响应机制
        经典的LDP技术
        
        以一定概率翻转布尔值
        """
        p = math.exp(self.epsilon) / (1 + math.exp(self.epsilon))
        
        if np.random.random() < p:
            return true_value
        else:
            return not true_value
    
    def rappor(self, value: int, domain_size: int) -> np.ndarray:
        """
        RAPPOR算法
        用于字符串/类别数据的LDP
        
        Args:
            value: 真实值（类别索引）
            domain_size: 类别总数
        """
        # 创建One-hot编码
        bloom = np.zeros(domain_size)
        bloom[value] = 1
        
        # 添加噪声
        p = 1 / (1 + math.exp(self.epsilon / 2))
        
        noisy = np.array([
            1 - b if np.random.random() < p else b
            for b in bloom
        ])
        
        return noisy
    
    def piecewise_mechanism(self, value: float, 
                           low: float, high: float) -> float:
        """
        分段机制
        用于数值型数据的LDP
        """
        # 归一化到[0,1]
        t = (value - low) / (high - low)
        t = max(0, min(1, t))  # 裁剪
        
        C = (math.exp(self.epsilon / 2) + 1) / (math.exp(self.epsilon / 2) - 1)
        
        # 定义分段概率密度
        l_t = (C + 1) / 2 * t - (C - 1) / 2
        r_t = l_t + C - 1
        
        # 采样
        if np.random.random() < (math.exp(self.epsilon / 2)) / (math.exp(self.epsilon / 2) + 1):
            # 从[l_t, r_t]均匀采样
            t_star = np.random.uniform(l_t, r_t)
        else:
            # 从[0, l_t) ∪ (r_t, 1]均匀采样
            if np.random.random() < l_t / (l_t + 1 - r_t):
                t_star = np.random.uniform(0, l_t)
            else:
                t_star = np.random.uniform(r_t, 1)
        
        # 反归一化
        return t_star * (high - low) + low


# 使用示例：本地差分隐私
if __name__ == "__main__":
    print("=== Local Differential Privacy ===\n")
    
    ldp = LocalDifferentialPrivacy(epsilon=2.0)
    
    # 随机响应示例
    true_answers = [True, False, True, True, False] * 100
    private_answers = [ldp.randomized_response(a) for a in true_answers]
    
    true_rate = sum(true_answers) / len(true_answers)
    private_rate = sum(private_answers) / len(private_answers)
    
    # 纠正偏差
    p = math.exp(2.0) / (1 + math.exp(2.0))
    corrected_rate = (private_rate - (1 - p)) / (2 * p - 1)
    
    print(f"True rate: {true_rate:.3f}")
    print(f"Private rate (raw): {private_rate:.3f}")
    print(f"Corrected rate: {corrected_rate:.3f}")
```

```python
        """
        计算每个样本的梯度
        需要使用支持此功能的框架（如Opacus）
        """
        # 这里简化示意

        pass
    
    def clip_and_accumulate(self, per_sample_grads):
        """
        裁剪并累积梯度
        """

        norms = [g.norm(2) for g in per_sample_grads]

        # 裁剪梯度
        clipped_grads = []
        for grad, factor in zip(per_sample_grads, factors):
            clipped_grads.append(grad * factor)
        
        # 累加梯度
        summed_grad = sum(clipped_grads)
        
        return summed_grad
    
    def add_noise(self, summed_grad):
        """
        添加高斯噪声实现差分隐私
        """
        noise_std = self.noise_multiplier * self.max_grad_norm
        noise = torch.randn_like(summed_grad) * noise_std
        
        return summed_grad + noise
    
    def step(self, loss):
        """
        执行一步DP-SGD更新
        """
        self.optimizer.zero_grad()
        
        # 1. 计算每样本梯度
        per_sample_grads = self.compute_per_sample_gradients(loss)
        
        # 2. 裁剪并累加
        summed_grad = self.clip_and_accumulate(per_sample_grads)
        
        # 3. 添加噪声
        noisy_grad = self.add_noise(summed_grad)
        
        # 4. 平均
        averaged_grad = noisy_grad / self.batch_size
        
        # 5. 更新参数
        for param, grad in zip(self.model.parameters(), [averaged_grad]):
            param.grad = grad
        
        self.optimizer.step()
    
    def compute_epsilon(self, epochs: int, delta: float) -> float:
        """
        计算给定训练设置的隐私预算
        
        使用 moment accountant 方法
        """
        # 简化计算，实际应使用精确的privacy accountant
        q = self.batch_size / self.dataset_size
        steps = epochs * self.dataset_size // self.batch_size
        
        # 使用高斯机制的隐私分析
        epsilon = math.sqrt(2 * math.log(1.25 / delta)) / self.noise_multiplier
        epsilon *= math.sqrt(steps) * q
        
        return epsilon
```
