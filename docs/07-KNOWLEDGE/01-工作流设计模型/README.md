## 关键概念

### 工作流网 (Workflow Net)

**定义** (van der Aalst, 1998):
一个Petri网 PN = (P, T, F) 是工作流网当且仅当：

- 存在唯一的源库所 i，没有输入弧
- 存在唯一的汇库所 o，没有输出弧
- 添加从 o 到 i 的弧后，网是强连通的

**正确性准则**:

1. **安全性 (Safeness)**：每个库所最多一个令牌
2. **活性 (Liveness)**：无死锁
3. **正确完成 (Proper Completion)**：完成时只有 o 有令牌

### 工作流模式 (Workflow Patterns)

**控制流模式**:

```
1. 顺序 (Sequence)
2. 并行分支 (Parallel Split)
3. 同步 (Synchronization)
4. 排他选择 (Exclusive Choice)
5. 简单合并 (Simple Merge)
...
43个标准模式
```

**数据模式**:

- 数据可见性
- 数据交互
- 数据传输

**资源模式**:

- 创建模式
- 推模式
- 拉模式

### Saga模式

**语义**: 长事务的补偿机制

```
正常流程: T1 → T2 → T3 → ... → Tn
补偿流程: 如果 T3 失败，执行 C2 → C1
```

**两种实现**:

1. **编排式 (Choreography)**：服务间事件驱动
2. **编舞式 (Orchestration)**：中央协调器控制
